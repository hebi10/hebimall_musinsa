# 헤더 UI 문서

## 대상 파일
- `src/app/_components/header/Header.tsx`
- `src/app/_components/header/Header.module.css`

## 현재 기준
- 데스크톱 헤더 본문 텍스트는 `.navLink`와 `.userLink`가 담당한다.
- 모바일 메뉴 텍스트는 `.mobileNavLink`, `.mobileUserLink` 등 별도 스타일을 사용한다.

## 2026-04-23 조정 사항
- 데스크톱 네비게이션 폰트 크기를 `0.79rem`에서 `0.87rem`으로 키웠다.
- 데스크톱 사용자 메뉴 폰트 크기를 `0.74rem`에서 `0.81rem`으로 키웠다.
- 데스크톱 헤더는 `960px` 이상부터만 노출하고, 그보다 좁은 폭은 모바일 햄버거 메뉴를 유지한다.
- `960px`부터 `1279px`까지 구간은 메인 메뉴와 사용자 메뉴만 보여 주고, 작은 노트북 폭에 맞춰 간격과 폰트를 줄인다.
- 데스크톱에는 보조 메뉴 `.secondaryNav`를 추가해 이전에 빠졌던 `추천`, `이벤트`, `리뷰`, `1:1문의`, `상품문의` 링크를 다시 노출한다.
- 보조 메뉴 `.secondaryNav`는 `1280px` 이상 큰 화면에서만 노출한다.
- 모바일 고객지원 목록에도 `1:1문의` 링크(`/cs/inquiry`)를 복구했다.

## 2026-04-24 컬러 메모
- 헤더 로고와 메인/모바일 링크 hover에서 버건디 포인트를 제거하고 검정 기준으로 맞췄다.
- 헤더는 별도 강조색 없이 기본 타이포 대비만 유지하는 쪽으로 정리했다.

## 2026-04-24 레이어 메모
- 헤더의 반투명 배경과 blur를 제거하고 `var(--surface-raised)` 기반의 불투명 배경으로 바꿨다.
- 상단 레이어 효과보다 명확한 배경/보더 구분을 우선하는 쪽으로 정리했다.

## 2026-05-12 링크 안정화
- 데스크톱/모바일 공통 메인 메뉴에서 메인 섹션 앵커 이동을 제거했다.
- `신상`은 `/recommend?filter=new`, `베스트`는 `/recommend?filter=review`, `세일`은 `/main/sale`, `브랜드`는 `/brand`로 연결한다.
- `/brand` 페이지를 추가해 헤더와 404의 브랜드 링크가 없는 페이지로 떨어지지 않게 했다.
- 404의 오래된 `/sale`, `/ranking` 링크는 실제 존재하는 `/main/sale`, `/recommend?filter=review`로 교체했다.

## 작업 시 주의
- 헤더는 한 줄 레이아웃을 유지해야 해서 폰트 크기 변경 시 `gap`과 `min-height` 영향도 함께 본다.
- 모바일 메뉴는 별도 인터랙션과 레이아웃을 쓰므로 데스크톱 헤더 조정과 분리해서 다룬다.
- 보조 메뉴를 건드릴 때는 `1280px` 미만에서 숨김 처리와 `960px`~`1279px` 구간 간격 축소값을 함께 봐야 레이아웃이 깨지지 않는다.
